## Локальный запуск
#### `npm i` — установка зависимостей
#### `npm run dev` — запуск версии для разработчика (по адресу #### `http://localhost:3000/`)
#### `npm run start` — запуск продакшн-версии и экспресс-сервера (по адресу #### `http://localhost:3000/`)
